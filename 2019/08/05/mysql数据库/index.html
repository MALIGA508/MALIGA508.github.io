mysql -u root -p 进去数据库

show databases； 显示所有数据库

create database；创建数据库

show tables； 显示所有数据表

use 数据库名； 表示进入数据库

创建数据表：create table ’数据表名 ’（项名 类型 . .，项名 类型 . .，……）；
auto_increment 设置主键
engine=innodb default charset=gbk; //设置字符集
primary key（‘字段名’）去重复
auto_increment 设置自增
timestamp default current_timestamp 创建时间
